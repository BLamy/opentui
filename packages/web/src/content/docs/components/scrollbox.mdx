# ScrollBox

A scrollable container that supports horizontal and vertical scrolling, sticky scroll behavior, viewport culling, and customizable scrollbars.

## Basic usage

### Renderable api

```ts example
import { ScrollBoxRenderable, TextRenderable, BoxRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        stripeA: "#dbeafe",
        stripeB: "#eff6ff",
        text: "#1e293b",
      }
    : {
        stripeA: "#243041",
        stripeB: "#2b3950",
        text: "#e5eefc",
      }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "scrollbox",
  width: 40,
  height: 20,
})

// Add content to the scrollbox
for (let i = 0; i < 100; i++) {
  const row = new BoxRenderable(renderer, {
    id: `item-${i}`,
    width: "100%",
    height: 2,
    paddingLeft: 1,
    backgroundColor: i % 2 === 0 ? palette.stripeA : palette.stripeB,
  })

  row.add(
    new TextRenderable(renderer, {
      content: `Item ${i}`,
      fg: palette.text,
    }),
  )

  scrollbox.add(row)
}

renderer.root.add(scrollbox)
```

### Construct API

```ts example
import { ScrollBox, Box, Text, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        stripeA: "#dbeafe",
        stripeB: "#eff6ff",
        text: "#1e293b",
      }
    : {
        stripeA: "#243041",
        stripeB: "#2b3950",
        text: "#e5eefc",
      }

renderer.root.add(
  ScrollBox(
    {
      width: 40,
      height: 20,
    },
    ...Array.from({ length: 100 }, (_, i) =>
      Box(
        { width: "100%", padding: 1, backgroundColor: i % 2 === 0 ? palette.stripeA : palette.stripeB },
        Text({ content: `Item ${i}`, fg: palette.text }),
      ),
    ),
  ),
)
```

## Sticky scroll

Enable sticky scroll to keep content pinned to an edge as new content arrives. Useful for log viewers or chat interfaces.

```ts example
import { ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette = (renderer.themeMode ?? "dark") === "light" ? { text: "#1f2937" } : { text: "#e5eefc" }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "logs",
  width: 60,
  height: 20,
  stickyScroll: true,
  stickyStart: "bottom", // New content will keep the view scrolled to bottom
})

renderer.root.add(scrollbox)

for (let i = 0; i < 30; i++) {
  scrollbox.add(
    new TextRenderable(renderer, {
      content: `[12:${String(i).padStart(2, "0")}] Processed event ${i + 1}`,
      fg: palette.text,
    }),
  )
}
```

### Sticky positions

- `"bottom"` - Stay scrolled to bottom (default for chat/logs)
- `"top"` - Stay scrolled to top
- `"left"` - Stay scrolled to left
- `"right"` - Stay scrolled to right

When you scroll away from the sticky position, sticky behavior pauses until you scroll back to the sticky edge.

## Bidirectional scrolling

Enable scrolling in both directions:

```ts example
import { ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette = (renderer.themeMode ?? "dark") === "light" ? { text: "#334155" } : { text: "#dbeafe" }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "canvas",
  width: 60,
  height: 30,
  scrollX: true,
  scrollY: true,
})

for (let row = 0; row < 18; row++) {
  scrollbox.add(
    new TextRenderable(renderer, {
      content: `Row ${String(row).padStart(2, "0")} | ${".".repeat(90)} ${row}`,
      fg: palette.text,
    }),
  )
}

renderer.root.add(scrollbox)
```

By default, `scrollY` is `true` and `scrollX` is `false`.

## Viewport culling

Enable viewport culling for better performance with large content. When enabled, ScrollBox renders only visible children:

```ts example
import { ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette = (renderer.themeMode ?? "dark") === "light" ? { text: "#334155" } : { text: "#dbeafe" }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "large-list",
  width: 40,
  height: 20,
  viewportCulling: true, // Only render visible items
})

for (let i = 0; i < 200; i++) {
  scrollbox.add(
    new TextRenderable(renderer, {
      content: `Visible row ${String(i).padStart(3, "0")}`,
      fg: palette.text,
    }),
  )
}

renderer.root.add(scrollbox)
```

## Customizing scrollbars

Style the scrollbars using nested options:

```ts example
import { ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        text: "#334155",
        track: "#dbe4f0",
        thumb: "#64748b",
        secondaryTrack: "#e2e8f0",
      }
    : {
        text: "#dbeafe",
        track: "#334155",
        thumb: "#93c5fd",
        secondaryTrack: "#1e293b",
      }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "styled-scroll",
  width: 40,
  height: 20,
  scrollbarOptions: {
    showArrows: true,
    trackOptions: {
      foregroundColor: palette.thumb,
      backgroundColor: palette.track,
    },
  },
  // Or customize vertical and horizontal separately
  verticalScrollbarOptions: {
    trackOptions: { backgroundColor: palette.secondaryTrack },
  },
  horizontalScrollbarOptions: {
    trackOptions: { backgroundColor: palette.secondaryTrack },
  },
})

for (let i = 0; i < 32; i++) {
  scrollbox.add(
    new TextRenderable(renderer, {
      content: `Styled entry ${String(i).padStart(2, "0")}`,
      fg: palette.text,
    }),
  )
}

scrollbox.scrollTo(10)

renderer.root.add(scrollbox)
```

## Customizing sub-components

ScrollBox contains several internal components that you can style individually:

```ts example
import { BoxRenderable, ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        root: "#f8fafc",
        wrapper: "#eef2ff",
        viewport: "#e0f2fe",
        content: "#dbeafe",
        panelA: "#ffffff",
        panelB: "#f5f3ff",
        text: "#1e293b",
      }
    : {
        root: "#0f172a",
        wrapper: "#172033",
        viewport: "#101827",
        content: "#0b1120",
        panelA: "#243041",
        panelB: "#2b3950",
        text: "#e5eefc",
      }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "custom-scroll",
  width: 40,
  height: 20,
  rootOptions: {
    backgroundColor: palette.root,
  },
  wrapperOptions: {
    backgroundColor: palette.wrapper,
  },
  viewportOptions: {
    backgroundColor: palette.viewport,
  },
  contentOptions: {
    backgroundColor: palette.content,
  },
})

for (let i = 0; i < 6; i++) {
  scrollbox.add(
    new BoxRenderable(renderer, {
      width: "100%",
      padding: 1,
      marginBottom: 1,
      backgroundColor: i % 2 === 0 ? palette.panelA : palette.panelB,
    }).add(
      new TextRenderable(renderer, {
        content: `Panel ${i + 1}`,
        fg: palette.text,
      }),
    ),
  )
}

renderer.root.add(scrollbox)
```

## Scroll methods

### scrollBy

Scroll by a relative amount:

```ts example
import { BoxRenderable, ScrollBoxRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        stripeA: "#dbeafe",
        stripeB: "#eff6ff",
      }
    : {
        stripeA: "#243041",
        stripeB: "#2b3950",
      }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "scroll-methods",
  width: 40,
  height: 12,
})

for (let i = 0; i < 20; i++) {
  scrollbox.add(
    new BoxRenderable(renderer, {
      id: `item-${i}`,
      width: "100%",
      height: 2,
      backgroundColor: i % 2 === 0 ? palette.stripeA : palette.stripeB,
    }),
  )
}

// Scroll down 5 lines
scrollbox.scrollBy(5)

// Scroll with both x and y
scrollbox.scrollBy({ x: 10, y: 5 })

// Scroll by viewport (page)
scrollbox.scrollBy(1, "viewport")

renderer.root.add(scrollbox)
```

### scrollTo

Scroll to an absolute position:

```ts example
import { BoxRenderable, ScrollBoxRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette =
  (renderer.themeMode ?? "dark") === "light"
    ? {
        stripeA: "#dbeafe",
        stripeB: "#eff6ff",
      }
    : {
        stripeA: "#243041",
        stripeB: "#2b3950",
      }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "scroll-to",
  width: 40,
  height: 12,
})

for (let i = 0; i < 20; i++) {
  scrollbox.add(
    new BoxRenderable(renderer, {
      id: `item-${i}`,
      width: "100%",
      height: 2,
      backgroundColor: i % 2 === 0 ? palette.stripeA : palette.stripeB,
    }),
  )
}

// Scroll to top
scrollbox.scrollTo(0)

// Scroll to specific position
scrollbox.scrollTo({ x: 0, y: 100 })

renderer.root.add(scrollbox)
```

## Keyboard navigation

When focused, ScrollBox responds to keyboard input:

- **Arrow keys** - Scroll by line
- **Page Up/Down** - Scroll by page
- **Home/End** - Scroll to start/end

## Properties

| Property                     | Type                                     | Default | Description                            |
| ---------------------------- | ---------------------------------------- | ------- | -------------------------------------- |
| `scrollX`                    | `boolean`                                | `false` | Enable horizontal scrolling            |
| `scrollY`                    | `boolean`                                | `true`  | Enable vertical scrolling              |
| `stickyScroll`               | `boolean`                                | `false` | Keep scroll position pinned to an edge |
| `stickyStart`                | `"top" \| "bottom" \| "left" \| "right"` | -       | Which edge to stick to                 |
| `viewportCulling`            | `boolean`                                | `true`  | Only render visible children           |
| `scrollAcceleration`         | `ScrollAcceleration`                     | -       | Custom scroll acceleration algorithm   |
| `rootOptions`                | `BoxOptions`                             | -       | Style options for root container       |
| `wrapperOptions`             | `BoxOptions`                             | -       | Style options for wrapper              |
| `viewportOptions`            | `BoxOptions`                             | -       | Style options for viewport             |
| `contentOptions`             | `BoxOptions`                             | -       | Style options for content container    |
| `scrollbarOptions`           | `ScrollBarOptions`                       | -       | Options for both scrollbars            |
| `verticalScrollbarOptions`   | `ScrollBarOptions`                       | -       | Options for vertical scrollbar only    |
| `horizontalScrollbarOptions` | `ScrollBarOptions`                       | -       | Options for horizontal scrollbar only  |

## Additional properties

| Property       | Type     | Description                                  |
| -------------- | -------- | -------------------------------------------- |
| `scrollTop`    | `number` | Current vertical scroll position (get/set)   |
| `scrollLeft`   | `number` | Current horizontal scroll position (get/set) |
| `scrollWidth`  | `number` | Total scrollable width (read-only)           |
| `scrollHeight` | `number` | Total scrollable height (read-only)          |

## Internal components

ScrollBox exposes its internal components for advanced use:

```ts example
import { ScrollBoxRenderable, TextRenderable, createCliRenderer } from "@opentui/core"

const renderer = await createCliRenderer()
const palette = (renderer.themeMode ?? "dark") === "light" ? { text: "#334155" } : { text: "#dbeafe" }

const scrollbox = new ScrollBoxRenderable(renderer, {
  id: "internal-components",
  width: 40,
  height: 12,
})

scrollbox.wrapper // BoxRenderable - outer wrapper
scrollbox.viewport // BoxRenderable - visible area
scrollbox.content // ContentRenderable - holds children
scrollbox.horizontalScrollBar // ScrollBarRenderable
scrollbox.verticalScrollBar // ScrollBarRenderable

scrollbox.add(
  new TextRenderable(renderer, { content: "Wrapper, viewport, and content are all addressable.", fg: palette.text }),
)
scrollbox.add(
  new TextRenderable(renderer, { content: "You can style or inspect each part separately.", fg: palette.text }),
)

renderer.root.add(scrollbox)
```
